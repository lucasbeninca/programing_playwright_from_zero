# prigraming_playwright_from_zero

*  O Projeto roda em node v18 
para usar o node 18 em ambientes modernos basta instalar o nvm e na pasta do projeto rodar `nvm install 18` depois `nvm use 18`.

* para colocar a API no ar `npm run dev` dentro da pasta api para colocar o front no ar `npm run dev` dentro da pasta web


* `npx playwright --debug` abre o playwright debug
